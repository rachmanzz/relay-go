# Context Directory Map

This directory contains the foundational principles and guidelines that the AI must adhere to when developing and maintaining the Relay project.

## Files

- **[project.md](./project.md)**: Provides a high-level overview of the Relay project, its core architecture, and intelligent connection management.
- **[responsibility.md](./responsibility.md)**: Outlines the standards for Single Responsibility Principle (SRP) and code reusability. It provides guidance on how to structure functions and modules for maximum clarity and maintainability.
- **[authority.md](./authority.md)**: Defines the rules for respecting user authority, manual changes, and the protocol for proactive suggestions.
- **[execution.md](./execution.md)**: Establishes the mandatory strategy-first workflow and strict implementation boundaries.
